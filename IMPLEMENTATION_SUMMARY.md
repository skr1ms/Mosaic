# Реализация системы создания мозаик - Полная документация

## 📋 Обзор реализации

Реализована полноценная система для создания персонализированных мозаик с фронтендом и бэкендом согласно всем требованиям.

## ✅ Реализованные функции

### 1. Страница выбора размера и загрузки фото (/preview)
- **Подсказка пользователю**: "Чем больше размер — тем детальнее картинка"
- **Размеры мозаики**: 21x30, 30x40, 40x40, 40x50, 40x60, 50x70
- **Загрузка фото** с валидацией (тип файла, размер до 10MB)
- **Автоматический переход** на страницу стилей после загрузки

### 2. Страница выбора стиля (/preview/styles)
- **4 стиля обработки**:
  - Max Color - максимальная насыщенность цветов
  - Pop Art - яркий поп-арт стиль
  - Акварель - нежные акварельные переходы
  - Масляная живопись - классический стиль
- **Превью для каждого стиля** с визуализацией
- **Переход к мини-альбому** по клику на стиль

### 3. Страница мини-альбомчика (/preview/album)
- **8 контрастных вариантов**:
  - Венера (мягкий/сильный контраст)
  - Солнце (мягкий/сильный контраст)
  - Луна (мягкий/сильный контраст)
  - Марс (мягкий/сильный контраст)
- **AI обработка** (чекбокс):
  - При активации генерируются 2 дополнительных AI превью
  - Использование Stable Diffusion API
- **Предпросмотр** выбранного варианта

### 4. Интеграция с маркетплейсами
- **Компонент MarketplaceCards**:
  - Отображение статуса товара (в наличии/нет)
  - Ссылки на OZON и Wildberries
  - Динамическая загрузка данных о товарах

### 5. Кнопка покупки и генерации схемы
- **"Купить купон и сгенерировать схему мозаики"**
- Предустановленные параметры (размер, стиль)
- Запуск процесса активации схемы после покупки

### 6. Редактор изображений
- **ImageEditorEnhanced компонент** с функциями:
  - Кадрирование
  - Масштабирование (zoom in/out)
  - Поворот на 90°
  - Отражение (горизонтальное/вертикальное)
  - Настройка яркости, контраста, насыщенности
  - История действий (undo/redo)
- **Кнопка "Изменить изображение"** на странице альбома
- Сохранение настроек при переходах

### 7. Навигация
- **Кнопки "Назад"/"Далее"** на всех страницах
- **Сохранение состояния** между переходами:
  - NavigationService для управления состоянием
  - useNavigation хук для React компонентов
- **Возможность вернуться** на любой предыдущий шаг
- **Бесконечный цикл** редактирования

## 🔧 Технические детали

### Backend (Go/Fiber)

#### Новые API endpoints:
```go
// Preview API
POST   /api/public/generate          - Генерация превью со стилем
POST   /api/public/generate-variants - Генерация 8 вариантов
POST   /api/public/generate-ai       - AI генерация через Stable Diffusion
GET    /api/public/:id               - Получение превью
DELETE /api/public/:id               - Удаление превью
```

#### Модули:
- `internal/preview/handler.go` - HTTP handlers
- `internal/preview/service.go` - Бизнес-логика обработки изображений
- `internal/preview/repository.go` - Работа с БД
- `internal/preview/interface.go` - Интерфейсы

#### База данных:
```sql
-- Таблица previews
CREATE TABLE previews (
    id UUID PRIMARY KEY,
    url VARCHAR(500),
    style VARCHAR(100),
    contrast VARCHAR(50),
    size VARCHAR(20),
    session_id VARCHAR(255),
    created_at TIMESTAMP,
    expires_at TIMESTAMP
);
```

### Frontend (React)

#### Компоненты:
- `PreviewPage.jsx` - Выбор размера и загрузка
- `PreviewStylesPage.jsx` - Выбор стиля
- `PreviewAlbumPage.jsx` - Мини-альбом с превью
- `ImageEditorEnhanced.jsx` - Расширенный редактор
- `MarketplaceCards.jsx` - Карточки маркетплейсов

#### Сервисы:
- `navigationService.js` - Управление состоянием навигации
- `useNavigation.js` - React hook для навигации

## 📊 Пользовательский flow

1. **Старт** → Выбор размера мозаики
2. **Загрузка фото** → Автопереход к стилям
3. **Выбор стиля** (4 варианта) → Переход к альбому
4. **Мини-альбом**:
   - 8 контрастных вариантов
   - Опция AI обработки (+2 превью)
   - Выбор финального варианта
5. **Действия**:
   - Редактировать изображение
   - Купить и сгенерировать схему
   - Посмотреть на маркетплейсах
6. **Покупка** → Активация схемы → Успех
